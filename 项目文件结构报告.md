# Campus Hub 项目文件结构报告

## 项目概述

Campus Hub 是一个基于 Vue 3 + Spring Boot 的校园论坛平台，采用现代化的前后端分离架构。项目包含完整的前端用户界面、后端API服务、数据库配置和用户认证系统。

**技术栈：**
- 前端：Vue 3.5.25, Vite 7.2.4, Element Plus 2.13.0, Pinia 3.0.4
- 后端：Spring Boot 3.1.7, MyBatis-Plus 3.5.7, MySQL
- 认证：JWT Token
- 样式：CSS3，响应式设计
- 特效：tsparticles 3.9.1, 交互效果组件

## 4. 核心功能模块

### 4.1 用户认证系统
- **前端**: AuthModal.vue, LoginView.vue, user.js (Pinia store)
- **后端**: AuthController.java, JwtAuthenticationFilter.java, JwtUtil.java
- **功能**: 登录, 注册, JWT Token管理, 权限验证

### 4.2 用户资料系统
- **前端**: UserProfilePage.vue, SideProfileCard.vue, UserCard.vue
- **后端**: UserController.java, UserService.java, UserDetailVO.java
- **功能**: 用户资料展示, 关注/粉丝关系管理

### 4.3 文章系统
- **前端**: ArticlesView.vue, ArticleList.vue
- **后端**: ArticleController.java, ArticleService.java, Article.java
- **功能**: 文章发布, 列表展示, 详情查看

### 4.4 搜索系统
- **前端**: SearchBar.vue, SearchResultPage.vue
- **后端**: SearchController.java, SearchResultVO.java
- **功能**: 全站搜索, 结果分类展示

### 4.5 话题系统
- **前端**: TopicsView.vue
- **后端**: TopicService.java, Topic.java, TopicMapper.java
- **功能**: 话题分类, 标签管理

### 4.6 特效系统
- **前端**: CursorBoom.vue, ClickExplosion.vue, GlobalClickEffect.vue
- **功能**: tsparticles粒子效果, 用户交互特效

- Spring Boot 3.1.7
- MyBatis-Plus 3.5.7 ORM
- JWT身份验证

- **前端**: 组件PascalCase, 文件kebab-case

## 8. 运行说明

### 8.1 后端运行
```bash
cd backend
./mvnw spring-boot:run
```

### 8.2 前端运行
```bash
cd frontend
npm install
npm run dev
```

### 8.3 生产构建
```bash
# 前端构建
cd frontend && npm run build

# 后端打包
cd backend && ./mvnw clean package
```
